# one thing to know before you start

timestamps are marked only on domains you allow

To keep out of your way, the on-page timestamp marker runs on an allowlist — not on every site.

  • localhost is enabled out of the box (http and https, any port)
  • to add a site: open it, click the %s icon in the toolbar, then hit + add <domain> (this tab) — or type a domain under // marked domains
  • remove a domain anytime with ✕ — marks disappear immediately

everything else works everywhere

  • popup converter: epoch → date and date → epoch, with copy buttons
  • right-click any selected timestamp or date → Convert timestamp ⇄ date
  • hover a marked timestamp for local / utc / iso / relative — click a value to copy
